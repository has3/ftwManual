How to pull / push content from git hub to local terminal

How to create a repo

~/mkdir ftwManual

cd ~/mkdir ftwManual

git init

touch README.md

git add (repo)

git status

git commit -m "insert a relevent message"

git status

How to clone a fork

git clone (repo)

git remote add (repo)

git fetch upstream

How to pull in files from git

git pull (repo)

git add (file to be added)

How to push file to git

git push
